Login & Authenticate

Python program for authenticating and access log-in. Once registered and login, users have access to a Notes application. Uses Python, Jinja2, Flask, HTML, CSS, & JavaScript.

Simple validation and database storing for passwords/username. Once validated access to Notepad with add and delete functionality.

Developed first with the Mobile First methodology, then for desktop. Compatible with all mobile device and with a simple user interface. On going project, future update on UI for aesthetics and appeal.

Responsive Personal Portfolio

Responsive Personal Portfolio Website using HTML, CSS, & JavaScript. Contains functioning pages such as: About, Work, & Contact sections. Includes a dark & light theme. Working contact section that automatically opens user default email & sends emails.

Developed first with the Mobile First methodology, then for desktop. Compatible with all mobile devices and with a beautiful and pleasant user interface.

404 Error Landing Page

Responsive error 404 page. Adaptive text animations. Utilizing CSS animations, floating text, & shadow effects. Functioning back redirect button to previous page visited.

Developed first with the Mobile First methodology, then for desktop. Compatible with all mobile devices and with a beautiful and pleasant user interface.
